MRKB UI Improvements
====================

Overview

MRKB UI Improvements is a small Foundry VTT module that customizes and improves parts of the user interface with lightweight JavaScript and CSS. Its primary changes focus on the players list and various UI panels (hotbar, sidebar, left controls, pause overlay, etc.).

Key features

- Adds player avatar and character avatar images to the Players list and shows both the user name and the character name.
- Character names are clickable and will open the character's sheet when the current user has sufficient permission on the character (permission >= 2).
- Provides a collection of stylesheet customizations that change fonts, layout, and visual details for hotbar, sidebar, left controls, pause screen, and players panel.

Installation

1. Open Foundry VTT and go to "Add-on Modules" → "Install Module".
2. Paste the module manifest URL and install:
   - Manifest: https://raw.githubusercontent.com/mrkb-fvtt-modules/fvtt-ui-improvements/refs/heads/master/module.json
3. Enable the module in your world configuration and (optionally) update load order so the styles apply as expected.

개요

MRKB UI Improvements는 Foundry VTT의 UI 일부를 경량의 JavaScript와 CSS로 사용자화하고 개선하는 작은 모듈입니다. 주요 변경점은 플레이어 목록과 여러 UI 패널(핫바, 사이드바, 왼쪽 컨트롤, 일시정지 오버레이 등)에 집중되어 있습니다.

주요 기능

- 플레이어 목록에 사용자 아바타와 캐릭터 아바타를 추가하고 사용자 이름과 캐릭터 이름을 모두 표시합니다.
- 캐릭터 이름은 클릭 가능하며 현재 사용자가 캐릭터에 대해 충분한 권한(permission >= 2)을 가지고 있으면 캐릭터 시트를 엽니다.
- 핫바, 사이드바, 왼쪽 컨트롤, 일시정지 화면, 플레이어 패널 등 여러 UI 요소를 변경하는 스타일 시트를 제공합니다.

설치 방법

1. Foundry VTT에서 "Add-on Modules" → "Install Module"로 이동합니다.
2. 모듈 매니페스트 URL을 붙여넣고 설치합니다:
   - 매니페스트: https://raw.githubusercontent.com/mrkb-fvtt-modules/fvtt-ui-improvements/refs/heads/master/module.json
3. 월드 설정에서 모듈을 활성화하고(필요시 스타일 적용을 위해 로드 순서를 조정) 사용합니다.

概要

MRKB UI Improvements は、Foundry VTT のユーザーインターフェースの一部を軽量な JavaScript と CSS でカスタマイズ・改善する小さなモジュールです。主にプレイヤー一覧や各種 UI パネル（ホットバー、サイドバー、左側コントロール、一時停止オーバーレイ等）に手を加えます。

主な機能

- プレイヤーリストにユーザーのアバターとキャラクターのアバターを追加し、ユーザー名とキャラクター名の両方を表示します。
- キャラクター名はクリック可能で、現在のユーザーがそのキャラクターに対して十分な権限（permission >= 2）を持っている場合、キャラクターシートを開きます。
- ホットバー、サイドバー、左側コントロール、一時停止画面、プレイヤーパネルなどの外観を変更するスタイルシートを提供します。

インストール

1. Foundry VTT の「Add-on Modules」→「Install Module」を開きます。
2. モジュールのマニフェスト URL を貼り付けてインストールします:
   - マニフェスト: https://raw.githubusercontent.com/mrkb-fvtt-modules/fvtt-ui-improvements/refs/heads/master/module.json
3. ワールド設定でモジュールを有効化し、必要に応じてスタイルが正しく適用されるようにロード順を調整します。
